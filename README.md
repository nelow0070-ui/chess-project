# ♟️ Chess Analysis Project

체스 기보(PGN)를 저장하고 Stockfish 엔진을 활용해
사용자의 실수를 분석하는 프로젝트입니다.

---

## 🚀 주요 기능

- PGN 게임 저장
- 수별 FEN 기록
- Stockfish 기반 최적 수 분석
- 실수 / 블런더 자동 판별
- 특정 포지션(FEN) 기반 검색

---

## 🧠 기술 스택

- Python
- SQLite
- python-chess
- Stockfish
- Flask (예정)
- HTML / JS (chessboard.js)
